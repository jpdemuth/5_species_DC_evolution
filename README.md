# 5_species_DC_evolution
* Analysis pipeline for inference of ancestral expression states across the phylogeny of 5 beetle species 
* Analysis_notebook.ipynb contains analysis pipeline for the data contained in the combined_data_5_species.tar.gz archive
* the r_analysis folder contains scripts, inputs and outputs for the ancestral state reconstructions and Pagel's lambda calculations
* sc_og_list.txt is the list of 5-way single copy orthologs necessary for ancestal state reconstruction.
